[ WHY BSL 1.1? ]

The Business Source License is designed to protect solo developers from corporate extraction while keeping the code open and inspectable.

It's better than proprietary:

  • You can always see the code.
  • You can modify it for personal use.
  • You can verify there's no surveillance, telemetry, or backdoors.
  • The project can't be rug-pulled — the code is already out there.

It's better than pure AGPL:

  • Clearer commercial terms (BSL is explicit about what's allowed).
  • No ambiguity about "network use" triggers.
  • Easier for companies to evaluate and approve.
  • Eventually becomes Apache 2.0, which is widely trusted.

It protects the work:

  • Prevents big companies from taking the code and offering "Elara Cloud" without contributing back.
  • Gives the solo developer leverage to negotiate if someone wants to commercialize it.
  • Ensures that if value is extracted, it's done fairly.
The Change Date: On January 1, 2029, Elara Core's license automatically converts to Apache 2.0 — a fully permissive open source license. At that point, anyone can do anything with it, including commercial use.

This means the project has a built-in expiration on restrictions. It won't stay locked down forever. Four years is enough time to establish the project, build credibility, and prevent immediate rip-offs. After that, it's fully open.

[ WHAT IS APACHE 2.0? ]

After the Change Date (2029-01-01), Elara Core becomes Apache 2.0 licensed. This is one of the most permissive and widely trusted open source licenses.

Under Apache 2.0, you can:

  • Use it commercially without restrictions.
  • Modify it and sell the modified version.
  • Incorporate it into proprietary products.
  • Offer it as a hosted service.
  • Do basically anything, as long as you include the original license and copyright notices.

Apache 2.0 is the same license used by Kubernetes, Android, TensorFlow, and thousands of other projects. It's industry-standard, lawyer-approved, and corporate-friendly.

The BSL-to-Apache-2.0 path means Elara Core will eventually join the ranks of truly open projects. The BSL period is just a temporary protective measure.
