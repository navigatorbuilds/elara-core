Built-in memory feature for ChatGPT Plus users. Automatically learns from conversations. Cloud-only.

Why Elara Core?

Privacy First

Everything stays on your machine. No API calls, no telemetry, no cloud dependencies. Your conversations, your data, your control.

Built for MCP

Not a retrofit. Designed from the ground up as an MCP server with 45 tools for semantic memory, mood tracking, reasoning, and more.

Beyond Simple Recall

Mood-aware responses, mistake tracking, reasoning trails, episodic memory, dream synthesis. It's not just memory—it's continuity.

Free Forever

BSD-1-Clause license. No usage limits, no paid tiers, no per-token pricing. Run it on your hardware, modify it however you want.

Instant Setup

pip install elara-core, add to Claude Desktop config, restart. No accounts, no API keys (except optional Gmail), no configuration hell.

Developer-Friendly

Clean Python codebase, 90 tests, documented architecture. Want a custom tool? Add it. Want different embeddings? Swap them. It's your server.

Where Elara Falls Short

  • Solo developer project. Not backed by a company or VC funding. Updates depend on one person's time.
  • Smaller community. mem0 and Letta have more users, more tutorials, more third-party integrations.
  • Newer codebase. Less battle-tested than alternatives. Bugs will surface.
  • No multi-user support. Built for single-user workflows. No memory sharing across teams (yet).
  • Python only. If you need JavaScript/TypeScript integration, mem0's API might be easier.
  • ChromaDB dependency. Requires running a local vector DB. Some alternatives offer hosted solutions.
  • MCP-specific. If you're not using Claude or an MCP-compatible client, this won't help you.
