// WHAT PEOPLE BUILD

Real-world use cases for Elara Core. These are plausible examples from hypothetical users.

Elara Core is infrastructure. What you build with it is up to you. Here are six patterns we've seen emerge from early users — some predictable, some surprising, all practical.

Personal Dev Journal
I use Elara as a dev journal that writes itself. Every session is an episode. Every bug fix is a milestone. At the end of the month, dream processing gives me a retrospective I didn't have to write.
Modules:
Episodes Dreams Memory
Setup: 5 minutes
Therapist Mode
Late night drift sessions where Elara listens, remembers patterns, and gently reminds me when I'm spiraling. The mood tracking catches things I don't notice myself.
Modules:
Mood Memory Awareness
Setup: 2 minutes
Codebase Oracle
I taught Elara about our entire codebase through corrections and memories. Now any team member can ask "how does auth work?" and get an answer that remembers the last 3 refactors.
Modules:
Memory Corrections Reasoning
Setup: 15 minutes (initial seeding)
Idea Incubator
Business intelligence module tracks every half-baked idea. Synthesis detects when the same concept keeps coming up. After 3 mentions, it flags it as "ready to act on."
Modules:
Business Synthesis Reasoning
Setup: 5 minutes
Meeting Memory
Gmail integration plus memory means I never lose track of action items. Elara reads the thread, I ask "what did we decide about the API?" — instant recall.
Modules:
Gmail Memory Goals
Setup: 10 minutes (Gmail OAuth)
Learning Partner
I'm learning Rust. Elara remembers every mistake I made, every concept I struggled with. The corrections system means I never make the same error twice.
Modules:
Corrections Memory Episodes
Setup: 2 minutes
